# A collection of example programs in JavaScript and Typescript

The `js` folder contains some examples to understand the
most important features of JavaScript

the `ts` folder contains come TypeScript examples


## Running it in Visual Studio Code (VSC)

Open this directory in VSC, click on the bottom left icon `><` and select "Reopen in Container".
Then, open a terminal window within VSC and run

```
npm install
```

This will (locally) install TypeScript inside the container as a node package.

## Compiling

To compile TypeScript code, either run `tsc` inside the `TS` directory, or
execute:

```
npm run build
```

in the root directory. To cleanup the files generated by TypeScript (`*.js` and `*.js.map`)
run:

```
npm run clean
```
